    ℹ️ About Folio Drop

    Folio Drop is an all-in-one PDF toolkit that runs almost entirely inside your browser tab. There's nothing to install and no account to create — drop a file in, pick a tool, and download the result. It bundles 15 tools covering the everyday things people need to do with PDFs, plus two optional AI-assisted features.

    ⚙️ How It Works

    Every tool except AI Summarize does its work with JavaScript libraries running directly in your browser — pdf-lib reads and rewrites PDF structure (merge, split, rotate, watermark, page numbers, delete/reorder pages), PDF.js renders pages to extract text and images, Mammoth.js converts Word documents, JSZip packages files, and html2canvas handles layout capture for Word-to-PDF conversion. Your file is read into memory by the browser, transformed, and handed back to you as a download — it's never transmitted anywhere.

    The two exceptions are AI Summarize and the optional AI Smart Split helper inside the Split tool. These extract the text content from your PDF locally, then send just that text (capped at roughly 18,000 characters) to a small serverless function, which forwards it to an AI model to generate a summary or suggest split points. See the Privacy section below for exactly what that involves.

    ✨ About the AI Features

    AI Summarize reads up to the first 40 pages of your PDF, extracts the text, and asks an AI model for 3–6 key bullet points plus a suggested filename.

    AI Smart Split (inside the Split tool) samples a short snippet from each page and asks the AI to identify natural breakpoints — new chapters, new invoices, a new topic starting — then pre-fills suggested page ranges for you to review before splitting.

    Both features call a serverless API endpoint that tries Groq first (fast inference, generous free tier) and can fall back to Gemini if configured. These only work when the app is deployed with an AI provider key set — opening the file directly won't have a server to answer that request.

    🔒 Privacy & Data
    ✅ 14 of 15 tools: 100% local. Merge, Split, Rotate, Compress, PDF↔Image, Images↔PDF, PDF↔Word, Page Numbers, Watermark, Delete/Reorder Pages, and Compare all process your files entirely in your browser's memory. Nothing is uploaded, and closing the tab discards everything.
    ⚠️ AI Summarize & AI Smart Split: the raw text extracted from your PDF (not the file itself) is sent to a server-side proxy and then to the AI provider (Groq, and Gemini as a fallback) to generate a response. Only these two AI-triggered actions leave your device, only the text you chose to summarize/split, and only when you click those specific buttons.

    No accounts, no tracking scripts, and no analytics are built into this app. If deployed as a Progressive Web App, a service worker caches the app's own files (HTML/CSS/JS/icons) so it can still open offline — this is separate from and unrelated to the AI features.
